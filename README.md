# ViralScan


A secure, containerised malware scanning web app using VirusTotal API with AI-powered explanations (Gemini).
## Initial planning:


TechStack

- Python - FastAPI (backend)
- Nginx (reverse proxy + security)
- JavaScript (frontend)


Using:
- VirusTotal API
- Gemini API
- AWS EC2 (deployment)

Additionally:
Docker (containerisation)
GitHub Actions (CI/CD)
